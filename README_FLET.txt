=== ИНСТРУКЦИЯ ПО ЗАПУСКУ ПРИЛОЖЕНИЯ ===

1. Установите зависимости:
   pip install -r requirements.txt

2. Запустите приложение:
   python main.py

=== СТРУКТУРА ПРОЕКТА ===

scripts/
├── main.py              - Главный файл приложения, точка входа
├── config.py            - Конфигурация (цвета, константы, настройки)
├── database.py          - Класс для работы с SQLite базой данных
├── models.py            - Модели данных (Child, Group) и вспомогательные функции
├── components.py        - Переиспользуемые UI компоненты
├── views/
│   ├── children_view.py - Представление для управления детьми
│   └── groups_view.py   - Представление для управления группами
└── requirements.txt     - Зависимости проекта

=== ВОЗМОЖНОСТИ ПРИЛОЖЕНИЯ ===

УПРАВЛЕНИЕ ГРУППАМИ:
- Добавление новых групп
- Редактирование информации о группах
- Удаление групп
- Просмотр статистики по группам (количество детей, мальчиков, девочек)

УПРАВЛЕНИЕ ДЕТЬМИ:
- Добавление новых детей с полной информацией
- Редактирование данных ребенка
- Удаление детей из базы
- Поиск детей по фамилии или имени
- Автоматический расчет возраста
- Назначение детей в группы

ИНТЕРФЕЙС:
- Боковое навигационное меню для переключения между разделами
- Таблицы с данными
- Формы для ввода/редактирования данных
- Диалоги подтверждения для критических действий
- Статистические карточки

=== ТЕХНИЧЕСКИЕ ДЕТАЛИ ===

- База данных: SQLite (файл kindergarten.db создается автоматически)
- UI Framework: Flet (Python)
- Архитектура: MVC-подобная структура с разделением на views, models, components
- Валидация данных на уровне UI и базы данных
